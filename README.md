# Mecanum Wheel Robot – Mechanical Design
## Overview

This project presents the mechanical design of a mecanum-wheel mobile robot
developed for indoor transport applications.

The robot uses a layered chassis architecture that separates
cargo storage, electronics, and drive components.
## Key Features

- Mecanum wheel drive for omnidirectional motion
- Modular layered chassis design
- Dedicated electronics compartment
- Cargo storage area on the upper levels
- Removable top lid and cargo storage for maintenance and hardware access****
